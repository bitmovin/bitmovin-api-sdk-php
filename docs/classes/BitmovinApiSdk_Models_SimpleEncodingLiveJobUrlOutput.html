Note that most protocols will require &#x60;credentials&#x60; to access the asset. Check the description below which ones are applicable. See below how to construct the URLs for the individual protocals/storage systems. **S3**: * &#x60;s3://&lt;my-bucket&gt;/path/&#x60; Authentication can be done via accesskey/secretkey or role-based authentication.
